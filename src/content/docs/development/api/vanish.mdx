## Vanish options

`VanishOptions` controls transition behavior (messaging, notifications, join/quit context flags).

## Disappear with options

<Tabs>
  <TabItem label="Java">

```java
import org.sayandev.sayanvanish.api.VanishOptions;
import org.sayandev.sayanvanish.api.VanishUser;

VanishUser user = VanishUser.getVanishUserFuture(player.getUniqueId()).get();
if (user == null) return;

VanishOptions options = new VanishOptions.Builder()
    .sendMessage(true)
    .notifyStatusChangeToOthers(false)
    .notifyJoinQuitVanished(true)
    .build();

user.disappear(options);
```

  </TabItem>
  <TabItem label="Kotlin">

```kotlin
import org.sayandev.sayanvanish.api.VanishOptions
import org.sayandev.sayanvanish.api.VanishUser

val user = VanishUser.getVanishUserFuture(player.uniqueId).get() ?: return

val options = VanishOptions.Builder()
    .sendMessage(true)
    .notifyStatusChangeToOthers(false)
    .notifyJoinQuitVanished(true)
    .build()

user.disappear(options)
```

  </TabItem>
</Tabs>
